# Orbit Stack Frontend Motion Demo

A cinematic, moving landing page concept inspired by modern AI platform sites
like Open WebUI.

This demo is intentionally fictional and reusable. It is not a clone of Open
WebUI, but it studies the same style ingredients: full-viewport motion, dark
technical atmosphere, short product copy, sticky minimal navigation, large
headlines, animated interface metaphors, and a command-line call to action.

## How Open WebUI Appears To Be Built

From the public page, Open WebUI appears to use a SvelteKit-style build because
its assets are served from `/_app/immutable/...`. The motion likely combines
large background media or animated graphics, layered CSS, component-level
transitions, and scroll-composed sections.

## Direction

Orbit Stack uses:

- a React canvas animation for the hero motion field
- CSS animations for orbital nodes and pulsing interface elements
- oversized high-contrast typography
- dark cinematic product storytelling
- reusable sections for platform, network, deploy, and notes

## Tech

- Vite
- React
- TypeScript
- Canvas
- CSS animations
- lucide-react icons

## Commands

From the repo root:

```bash
npm run dev:orbit
npm run build:orbit
```

From this demo folder:

```bash
npm run dev
npm run build
```
