# Khara Collective Frontend Concept

Modern frontend-only rebuild concept for Khara Collective, a boutique curated
gifting studio.

## Audience

The concept is aimed at people who need thoughtful gifts without managing every
detail themselves:

- corporate clients and people teams
- realtors and closing gift buyers
- event planners
- founders or operators sending client appreciation gifts

## Direction

The design avoids a SaaS or dashboard feel. It uses a warm editorial hero,
large serif typography, product-forward photography, and concise service
sections to make the business feel personal, tasteful, and capable.

The page emphasizes:

- curated client, team, event, and real estate gifting
- local and relationship-focused sourcing
- done-for-you logistics
- a simple quote/inquiry path

## Tech

- Vite
- React
- TypeScript
- CSS
- lucide-react icons

## Commands

From the repo root:

```bash
npm run dev:khara
npm run build:khara
```

From this demo folder:

```bash
npm run dev
npm run build
```

## Reuse Notes

This is a good base for other local service, boutique retail, gifting, events,
florist, realtor, concierge, or hospitality sites where the owner needs a more
human, editorial presence instead of a generic template.
